For information on how the chat room works, go to this link:
https://medium.com/swlh/lets-write-a-chat-app-in-python-f6783a9ac170
In order to create a chat room, user needs to follow the following steps:

1. Run chat_serv.py by using the command: python chat_serv.py
2. Open a new terminal window and run chat_clnt.py by using the command: python chat_clnt.py
3. Enter
	host: 127.0.0.1
	port: 8000
